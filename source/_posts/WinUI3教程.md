## 0.前置知识
- GUI
    - 即Graphic User Interface，它提供用户友好的图像操作界面
- CLI
    - 即Command-Line Interface，命令行交互，一般开发者在使用

- Windows SDK
    - 即Windows Software Development Kit，微软官方给开发者提供的开发工具，相当于是访问Windows的API
