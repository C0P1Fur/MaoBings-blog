## 1.排序
#### 1.1快速排序
性质：平均时间复杂度为O(nlogn)，最坏时间复杂度为O(n^2)

主要思想
- 最核心的思路：分而治之，将大问题分成一个个小问题
- 具体实现
    - 定义一个左边界和右边界，圈出范围
    - 在范围里随机取数，一般取中间的，设为x
    - 将小于x的数排在x左边，大于的排在右边
    - 此时范围被分为两部分：左半小，右半大
    - 继续对左半与右半执行一遍这个过程

代码范例，最核心的代码段是quick_sort
```
#include<iostream>

using namespace std;

const int N = 1e6 + 10; // + 10防止数组越界，是一种缓冲区
int n;
int a[N];

void quick_sort(int * a, int l, int r) //l表示left，r表示right
{
    if (l >= r) return;
    
    /**
    * l + r >> 1表达式与(l + r) / 2效果一样
    * 但是更快而且更简便
    * 
    * l + r >> 1默认是向下取整
    * 使用 l + r + 1 >> 1可以向上取整
    * 
    * 同时，将x去中间的一个数可以避免死循环
    */
    int x = a[l + r >> 1], i = l - 1, j = r + 1;
    while (i < j)
    {
        do i++; while (a[i] < x);
        do j--; while (a[j] > x);
        if (i < j) swap(a[i], a[j]);
    }
    
    /**
    * 这里要么写l, j 要么写l，i - 1
    * 因为最后要将数组一分为二
    * i 与 j的大小就有两种情况
    * - i == j，此时两种方法的效果是一样的
    * - j < i，此时j指向数组的左半部分，就不能向左偏移了，i同理
    */
    quick_sort(a, l, j);
    quick_sort(a, j + 1, r);
}

int main(void)
{
    scanf("%d", &n); //用scanf读取数据比cin要快一点
    for (int i = 0; i < n; i++) scanf("%d", &a[i]);
    
    quick_sort(a, 0, n - 1);
    
    for (int i = 0; i < n; i++) printf("%d ", a[i]);
    
    return 0;
}
```

#### 1.1(2) 快速选择排序
选择一遍递归，而不是两边都递归
所以时间复杂度为n + n / 2 + n / 4 + n / 8 + ... < 2n
所以时间复杂度为O(n)

#### 1.2 归并排序
性质：时间复杂度稳定为nlogn

思想
- 将数组不断划分，分为很小的部分
- 对小部分进行排序，这下我们得到了两个顺序数组
- 将两个小部分合成，又得到了新的顺序数组
- 不断合成，最后结果为原来的大数组

代码实现
```
void merge_sort(int * a, int l, int r)
{
    if (l >= r) return;
    int mid = l + r >> 1;
    
    // 切分操作
    merge_sort(a, l, mid), merge_sort(a, mid + 1, r);
    
    //排序操作
    int k = 0, i = l, j = mid + 1;
    while (i <= mid && j <= r)
    {
        if (a[i] <= a[j]) t[k++] = a[i++];
        else t[k++] = a[j++];
    }
    while (i <= mid) t[k++] = a[i++];
    while (j <= r) t[k++] = a[j++];
    
    for (i = 0, j = l; j <= r; i++, j++) a[j] = t[i];
}
```

## 2.二分
#### 2.1 整数二分
有单调性一定可以二分，二分不一定需要单调性
二分思想的本质
- 现有一数组，有一值，在数组里找值
- 通过对数组中点性质的比较，可以缩小值的范围
- 不断缩小，最后精确找到值

代码实现
```
// 最终结果：l == r，指向答案
// 特点：这种模板不存在找不到的情况，找不到需要按照题意单独判断
// 特点：一定可以把符合性质的范围二分出来
// 选择：当check为true使，右边界收缩时，就用第一种，反之第二种
int bsearch1(int l, int r)
{
    while (l < r)
    {
        mid = l + r >> 1;
        if (check()) r = mid;
        else l = mid + 1;
    }
    return l;
}

int bsearch2(int l, int r)
{
    while (l < r)
    {
        mid = l + r + 1 >> 1;
        if (check()) l = mid;
        else r = mid - 1;
    }
}
```

#### 2.2浮点数二分
和整数二分类似，只不过**不用考虑向下或是向上取整问题**
代码实现
```
double bsearch_3(double l, double r)
{
    const double eps = 1e-6;   // eps 表示精度，取决于题目对精度的要求
    while (r - l > eps)
    {
        double mid = (l + r) / 2;
        if (check(mid)) r = mid;
        else l = mid;
    }
    return l; //返回l或者r的一个
}

bool cmp(vector<int> a, vector<int> b)
{
    if (a.size() > b.size()) return true;

    else{

        }

// 不取minus是因为内置函数有这个名
vector<int> sub(vector<int> a, vector<int> b)
{
    vector<int> c;
    for (int i = 0, t = 0; i < a.size(); i++) // a, b, c共用i的原因是，它们的定位都是相同的
    {
        t = a[i] - t;
        if (i < b.size()) t -= b[i];
        
        c.push_back((t + 10) % 10);
        if (t < 0) t = 1;
        else t = 0;
    }
    
    while (c.size() > 1 && c.back() == 0) c.pop_back();
    return c;
}

## 4.前缀和与差分
#### 4.1 前缀和
```
S[i] = a[1] + a[2] + ... a[i]
a[l] + ... + a[r] = S[r] - S[l - 1]
```

为什么需要前缀和？
当需要重复计算区间[l, r]之间的数据和的时候，每次计算的时间复杂度是O(n)
但计算一次前缀和数列后，每一计算的时间复杂度为O(1)

#### 4.2 前缀和矩阵
```
S[i, j] = 第i行j列格子左上部分所有元素的和
以(x1, y1)为左上角，(x2, y2)为右下角的子矩阵的和为：
S[x2, y2] - S[x1 - 1, y2] - S[x2, y1 - 1] + S[x1 - 1, y1 - 1]
```

补充说明：矩阵的和，等于从某个坐标开始，左上角所有数字的和

#### 4.3 差分
```
给区间[l, r]中的每个数加上c：B[l] += c, B[r + 1] -= c
只需要一个insert(l, r, c)操作
```

差分就是前缀和的逆运算，也就是给定数组，算差分数组，数组就是差分数组的前缀和数组
使用情况：在某个区间给所有数加上一个数

差分不需要额外操作来构建，只需构想一个空差分组，用insert方法在[i, i]处加上ai即可构建

#### 4.4 差分矩阵
```
给以(x1, y1)为左上角，(x2, y2)为右下角的子矩阵中的所有元素加上c：
S[x1, y1] += c, S[x2 + 1, y1] -= c, S[x1, y2 + 1] -= c, S[x2 + 1, y2 + 1] += c
同样只需要一个insert操作
```

## 5.双指针算法
思想：涉及到两个指针的操作
有几种情况：
- 指向两个不同的序列
    - 归并排序，两个指针分别确定操作位置
- 指向一个序列
    - 快速排序，两个指针确定区间

双指针算法的具体模板
```
for (int i = 0, j = 0; i < n; i++)
{
    while (j < i && check(i, j)) j++;

    // 每道题目的具体逻辑
}
```
i与j移动的次数不超过n，因此时间复杂度为O(n)

双指针问题如何思考
先思考如何暴力解决问题
再在暴力解决问题的基础上发现单调性
如最长不重复子序列，左右指针得同时向前移动，这样就可以将复杂度降到2n
```
#include <iostream>

using namespace std;

const int N = 1e5 + 10;
int a[N];
char s[N]; // 存储数字的重复状态
int n, res;

int main(void)
{
    scanf("%d", &n);
    for (int i = 0; i < n; i++)
        scanf("%d", &a[i]);
    
    for (int i = 0, j = 0; i < n; i++) // i与j区间内必须保证没有重复元素
    {
        s[a[i]]++;
        while (j < i && s[a[i]] > 1)
        {
            s[a[j]]--;
            j++;
        }
        
        res = max(res, i - j + 1);
    }
    
    cout << res;
}
```

使用注意：
必须时刻保证指针不越界，所以要在必要的地方加上限定条件

## 7.离散化
特指整数离散化
将大坐标映射到小坐标的过程
然后每次对于大坐标的操作，都需要映射到小坐标才行

实现离散化的思路
1. 去除数组中重复元素
    - 需要对数组去重
2. 算出大坐标对应的小坐标
    - 需要二分查找

实现的具体代码
```
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

const int N = 300010;
int a[N], s[N];
int n, m;

vector<int> alls;
vector<pair<int, int>> adds, query;

int finds(int x) // 映射操作
{
    int mid, l = 0, r = alls.size() - 1;
    while (l < r)
    {
        mid = l + r >> 1;
        if (alls[mid] >= x) r = mid;
        else l = mid + 1;
    }

    return l + 1; // 将下标向前偏移一位，方便之后求前缀和
}

int main(void)
{
    scanf("%d %d", &n, &m);
    
    for (int i = 0; i < n; i++)
    {
        int x, c;
        scanf("%d %d", &x, &c);
        
        alls.push_back(x); // 因为 x, l, r都是大坐标，所以都要添加到表里
        adds.push_back({x, c});
    }
    
    for (int i = 0; i < m; i++)
    {
        int l, r;
        scanf("%d %d", &l, &r);
        
        query.push_back({l, r});
        alls.push_back(l), alls.push_back(r); // 因为 x, l, r都是大坐标，所以都要添加到表里
    }
    
    sort(alls.begin(), alls.end());
    alls.erase(unique(alls.begin(), alls.end()), alls.end()); // 离散化
    
    for (auto item : adds) // 加入数字
    {
        a[finds(item.first)] += item.second;
    }
    for (int i = 1; i <= alls.size(); i++) s[i] = s[i - 1] + a[i]; // 求前缀和
    
    for (auto item : query)
    {
        int sum = s[finds(item.second)] - s[finds(item.first) - 1];
        cout << sum << endl;
    }
    
}
```

## 8.区间合并
1. 按照区间左端点排序
2. 选定维护区间
3. 查看区间内是否有其他区间的左端点，有就更新
```
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

const int N = 100010;
int n;
vector<pair<int, int>> segs, res;

int main(void)
{
    scanf("%d", &n);
    for (int i = 0; i < n; i++)
    {
        int l, r;
        cin >> l >> r;
        segs.push_back({l, r});
    }
    
    sort(segs.begin(), segs.end());
    int st = -2e9, ed = -2e9;
    for (auto seg : segs)
    {
        if (ed < seg.first) // 区间左端点不在范围内，更新区间
        {
            if (ed != -2e9) res.push_back({st, ed});
            st = seg.first;
            ed = seg.second;
        }
        else // 在区间内
            ed = max(ed, seg.second);
    }
    
    if (ed != -2e9) res.push_back({st, ed});
    
    cout << res.size() << endl;
}
```

## 9.第一章基础算法总结
#### 9.1 for循环的用法
1. 可以用来控制循环次数
    - `for (int i = 0; i < n /* 这个判断条件与i有关 */ ; i++)`
2. 可以用来计数
    - `for (int c = 0; x != 0 /* 这个判断条件与i无关 */ ; c++)`
    - 此代码等同`while (x != 0)`，只不过c起到了一个计数的作用
3. 可以用来控制边界条件
    - `for (int i = 0, j = 0; i < n && j < m; i++)`
    - 用来确保两个指针的范围，确保执行中 i < n，且 j < m
    - 此代码等同`while (i < n && j < m)`，只不过每次循环完会改变i位置

#### 10.1 单链表代码
```
#include <iostream>

using namespace std;

const int N = 1e5 + 10;
int n;

// e与ne通过下标关联起来，所以不用单独开一个结构来存储
// e表示存的数据，ne表示下一个节点
// head是头指针，不存储数据
// 可以将idx想象成指向新添加成员的内存指针
int e[N], ne[N];
int head;
int idx;

void init(void)
{
    head = -1;
    idx = 0;
}

// 删除第k个元素指向的元素
// 直接将k处的指针指向下下一个元素
void removeK(int k)
{
    ne[k] = ne[ne[k]];
}

// 移除head指针指向的元素
void removeH(void)
{
    head = ne[head];
}

// 在第k个元素后添加成员
void addK(int k, int x)
{
    e[idx] = x;
    ne[idx] = ne[k];
    // 前两行可以这样理解：先新分配内存，把这个节点需要存储的内容存下来
    
    ne[k] = idx;
    idx++;
}

// 将head指针指向新加的元素
void addH(int x)
{
    e[idx] = x;
    ne[idx] = head;
    // 可以像上面的操作那样理解
    
    head = idx;
    idx++;
}

int main(void)
{
    init();
    
    scanf("%d", &n);
    char op[2];
    int k, x;
    for (int i = 0; i < n; i++)
    {
        cin >> op;
        if (op[0] == 'H')
        {
            cin >> x;
            addH(x);
        }
        else if (op[0] == 'D')
        {
            cin >> k;
            if (k == 0)
                removeH();
            else
                removeK(k - 1);
        }
        else
        {
            cin >> k >> x;
            addK(k - 1, x);
        }
    }
    
    for (int i = head; i != -1; i = ne[i])
        cout << e[i] << " ";
}
```

#### 10.2 双链表代码
```
#include <iostream>
#include <string>

using namespace std;

const int N = 1e5 + 10;
int n;
// 和双链表的逻辑相同，只不过在结构体里，多添加了一个向前指的指针
int l[N], r[N], e[N], idx;

void init(void)
{
    r[0] = 1;
    l[1] = 0;
    idx = 2;
}

// 在k的右侧添加数据
void adds(int k, int x)
{
    e[idx] = x;
    l[idx] = k;
    r[idx] = r[k];
    
    l[r[k]] = idx;
    r[k] = idx;
    idx++;
}

// 删除第k个数
void removes(int k)
{
    r[l[k]] = r[k];
    l[r[k]] = l[k];
}

int main(void)
{
    scanf("%d", &n);
    init();
    
    string op;
    int k, x;
    for (int i = 0; i < n; i++)
    {
        cin >> op;
        if (op == "L")
        {
            cin >> x;
            adds(0, x);
        }
        else if (op == "R")
        {
            cin >> x;
            adds(l[1], x);
        }
        else if (op == "D")
        {
            cin >> k;
            removes(k + 1);
        }

        else
        {

        }

## 13.trie树
基本用法：高效的存储和查找英文字符串的数据结构
```
#include <iostream>

using namespace std;

const int N = 1001000;
char son[N][26]; // 记录子节点与父节点的关系
int cnt[N]; // 记录每个节点存储的单词数
int idx;
int n;

void insert(string & x)
{
    int p = 0; // 从根节点开始查找
    for (auto item : x)
    {
        if (!son[p][item - 'a']) son[p][item - 'a'] = ++idx; // 如果为空节点则创建
        p = son[p][item - 'a'];
    }
    cnt[p]++;
}

int find(string & x)
{
    int p = 0;
    for (auto item : x)
    {
        if (!son[p][item - 'a']) // 如果走不动就直接返回
            return 0;
        p = son[p][item - 'a'];
    }
    return cnt[p]; // 返回此节点结束的单词数
}

int main(void)
{
    scanf("%d", &n);
    for (int i = 0; i < n; i++)
    {
        string op, x;
        cin >> op >> x;
        if (op == "I") insert(x);
        else if (op == "Q") printf("%d\n", find(x));
    }
}
```

## 14.并查集
**使用范围**
1. 查询某个对象是否属于某个集合
2. 合并两个集合

**基本原理**
每个集合用树来表示，树根的编号就是集合的编号。使用p[x]存储每个元素的父节点
```
#include <iostream>

using namespace std;

const int N = 1000100;
int p[N];
int n, m;

// 找到x的根节点
int find(int x)
{
    if (!p[x] == x) p[x] = find(p[x]);
    return p[x];
}

int main(void)
{
    scanf("%d %d", &n, &m);
    for (int i = 1; i <= n; i++) p[i] = i;
    
    for (int i = 0; i < m; i++)
    {
        char op[2];
        int x, y;
        cin >> op >> x >> y;
        if (op[0] == 'M') p[find(x)] = find(y);
        else if (op[0] == 'Q')
        {
            if (find(x) == find(y)) printf("Yes\n");
            else printf("No\n");
        }
    }
}
```
